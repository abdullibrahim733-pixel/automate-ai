# ⚡ AutoMate AI

**Automation guides and AI tool reviews** — helping you work smarter, not harder.

## 🚀 Live Site

[https://automate-ai-ruby.vercel.app](https://automate-ai-ruby.vercel.app)

## 📋 Overview

AutoMate AI is an SEO-optimized content site focused on:
- **AI Tool Reviews** — Hands-on reviews of ChatGPT, Claude, Copilot, and more
- **Automation Guides** — Step-by-step workflows to automate repetitive tasks
- **Comparisons** — Head-to-head AI tool comparisons
- **AI Productivity** — Tips to get more done with AI assistance

## 🏗️ Tech Stack

- **Pure HTML/CSS/JS** — No framework, no build step, zero dependencies
- **Mobile-first responsive design** — 3 breakpoints
- **SEO optimized** — JSON-LD structured data, Open Graph, sitemap, RSS feed
- **Hosted on Vercel** — Auto-deploys from GitHub on every push

## 🔄 Auto-Deploy

Every push to the `main` branch triggers an automatic deployment to Vercel.
